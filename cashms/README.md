# Run and deploy your app

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. (Optional) Set `VITE_API_BASE_URL` in [.env.local](.env.local) if your backend is not running on `http://localhost:5001/api`
3. Run the app:
   `npm run dev`
